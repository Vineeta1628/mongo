What is MongoDB Compass?

The official GUI for MongoDB

💡 MongoDB Compass

MongoDB Compass is the official graphical user interface (GUI) for MongoDB. It allows you to visually explore your data, run queries, optimize performance, and manage your databases - all without writing a single line of code!

🔍
Visual Query Builder
Build queries visually without writing code - perfect for beginners
📊
Schema Analysis
Understand your data structure with automatic schema detection
⚡
Performance Insights
Analyze query performance with explain plans and index recommendations
📝
CRUD Operations
Create, read, update, and delete documents with a friendly interface
🧪
Aggregation Pipeline
Build complex aggregation pipelines with visual stage-by-stage editing
🔐
Secure Connections
Connect securely with SSL/TLS and SSH tunneling support
⚙️

System Requirements

Minimum specifications needed

Windows

  • OS: Windows 10 or later (64-bit)
  • RAM: 4 GB minimum
  • Disk: 500 MB free space
  • Processor: 64-bit processor

macOS

  • OS: macOS 10.12 or later
  • RAM: 4 GB minimum
  • Disk: 500 MB free space
  • Chip: Intel or Apple Silicon (M1/M2)

Linux

  • OS: Ubuntu 18.04+, RHEL 7+, or equivalent
  • RAM: 4 GB minimum
  • Disk: 500 MB free space
  • Desktop: GNOME or KDE required

Windows Installation

Step-by-step guide for Windows

1
Download MongoDB Compass

Visit the official MongoDB download page and download the Windows installer.

Download Compass for Windows

Screenshot: MongoDB Compass download page showing Windows installer option

File: mongodb-compass-x.xx.x-win32-x64.exe (~150 MB)
💡 Tip The installer automatically detects your Windows version. Choose the default "Download" button for the full version with all features.
2
Run the Installer
  1. Locate the downloaded .exe file in your Downloads folder
  2. Double-click the installer to launch it
  3. If prompted by Windows Defender, click "Run anyway" or "More info" → "Run anyway"

Screenshot: Windows installer welcome screen with "Install" button

3
Accept License Agreement
  1. Read the MongoDB Compass license agreement
  2. Check "I accept the terms in the License Agreement"
  3. Click "Next" to continue

Screenshot: License agreement screen with checkbox

4
Choose Installation Type

Select your installation preferences:

  • Complete: Installs all features (recommended for most users)
  • Custom: Choose specific components and installation location

For beginners, select "Complete" and click "Next".

Screenshot: Installation type selection screen

5
Install
  1. Review your installation settings
  2. Click "Install" to begin the installation
  3. Wait for the installation to complete (1-2 minutes)
  4. Click "Finish" when done

Screenshot: Installation progress bar

✅ Installation Complete! MongoDB Compass is now installed on your Windows system. You can find it in your Start Menu.
🍎

macOS Installation

Step-by-step guide for Mac

1
Download MongoDB Compass

Visit the official MongoDB download page and download the macOS installer.

Download Compass for macOS

Screenshot: Download page with macOS .dmg file option

File: mongodb-compass-x.xx.x-darwin-x64.dmg (~150 MB)
💡 Apple Silicon Users If you have an M1 or M2 Mac, download the "macOS ARM64" version for better performance. Intel Macs should use the standard macOS version.
2
Open the DMG File
  1. Locate the downloaded .dmg file in your Downloads folder
  2. Double-click the DMG file to mount it
  3. A new window will open showing the MongoDB Compass icon

Screenshot: DMG window with MongoDB Compass icon and Applications folder

3
Drag to Applications
  1. Drag the MongoDB Compass icon to the Applications folder
  2. Wait for the copy to complete
  3. Eject the DMG file from the sidebar

Screenshot: Dragging MongoDB Compass to Applications folder

4
First Launch (Security)
  1. Open Applications folder
  2. Find MongoDB Compass and double-click it
  3. If you see a security warning, click "Open"
  4. Alternatively: Go to System Preferences → Security & Privacy → Click "Open Anyway"

Screenshot: macOS security dialog with "Open" button

⚠️ macOS Gatekeeper The first time you open Compass, macOS may warn you that it's from an unidentified developer. This is normal. Click "Open" to proceed.
Alternative: Install via Homebrew

If you have Homebrew installed, you can install Compass with one command:

brew install --cask mongodb-compass

This will automatically download and install MongoDB Compass.

🐧

Linux Installation

Step-by-step guide for Ubuntu/Debian and RHEL/Fedora

Ubuntu / Debian Installation

Method 1: Download .deb Package

  1. Visit MongoDB Download Page
  2. Select "Ubuntu 64-bit (.deb)" from the dropdown
  3. Click "Download"

Method 2: Terminal Installation

# Download the package wget https://downloads.mongodb.com/compass/mongodb-compass_1.42.1_amd64.deb # Install with dpkg sudo dpkg -i mongodb-compass_1.42.1_amd64.deb # Install dependencies if needed sudo apt-get install -f
💡 Note Replace the version number (1.42.1) with the latest version from the download page.
RHEL / Fedora Installation

Method 1: Download .rpm Package

  1. Visit MongoDB Download Page
  2. Select "Red Hat 64-bit (.rpm)" from the dropdown
  3. Click "Download"

Method 2: Terminal Installation

# Download the package wget https://downloads.mongodb.com/compass/mongodb-compass-1.42.1.x86_64.rpm # Install with yum/dnf sudo yum install mongodb-compass-1.42.1.x86_64.rpm # OR for Fedora sudo dnf install mongodb-compass-1.42.1.x86_64.rpm
🚀
Launch MongoDB Compass

Option 1: Application Menu

  • Open your application menu (GNOME Activities, KDE Application Launcher, etc.)
  • Search for "MongoDB Compass"
  • Click the icon to launch

Option 2: Terminal

mongodb-compass

First Launch & Setup

Getting started with MongoDB Compass

1
Welcome Screen

When you first launch MongoDB Compass, you'll see the welcome screen.

Screenshot: MongoDB Compass welcome screen with connection options

💡 Privacy Settings Compass may ask permission to collect usage statistics. You can choose to enable or disable this - it's completely optional.
2
Connect to MongoDB

For Local MongoDB:

  • Default connection string: mongodb://localhost:27017
  • Click "Connect" to connect to your local MongoDB instance
mongodb://localhost:27017

For MongoDB Atlas (Cloud):

  1. Get your connection string from Atlas
  2. Paste it into the connection field
  3. Click "Connect"

Screenshot: Connection screen with connection string input field

3
Explore the Interface

Once connected, you'll see the main Compass interface:

  • Left Sidebar: List of databases and collections
  • Top Bar: Connection info and navigation
  • Main Panel: Data view, query builder, and tools
  • Tabs: Documents, Aggregations, Schema, Explain Plan, Indexes

Screenshot: MongoDB Compass main interface showing databases and collections

✅ You're All Set! MongoDB Compass is ready to use. You can now explore databases, run queries, and manage your data visually!

Troubleshooting

Common issues and solutions

Issue: Cannot Connect to MongoDB

Symptoms: Connection timeout or "Cannot connect to server" error

Solutions:

  1. Make sure MongoDB server is running on your machine
  2. Check the connection string (default: mongodb://localhost:27017)
  3. Verify MongoDB is listening on port 27017: netstat -an | grep 27017
  4. Check firewall settings - MongoDB port may be blocked
Issue: Compass Won't Open (Windows)

Symptoms: Application crashes or doesn't start

Solutions:

  1. Run as Administrator (right-click → "Run as administrator")
  2. Disable antivirus temporarily to test
  3. Check Windows Event Viewer for error details
  4. Reinstall with latest version
Issue: macOS Security Warning

Symptoms: "Cannot be opened because it is from an unidentified developer"

Solution:

  1. Open System Preferences → Security & Privacy
  2. Click the lock icon to make changes
  3. Look for the message about MongoDB Compass
  4. Click "Open Anyway"
Issue: Linux Missing Dependencies

Symptoms: Error about missing libraries or dependencies

Solution:

# Ubuntu/Debian sudo apt-get install -f # RHEL/Fedora sudo yum install mongodb-compass-dependencies
